> `Python` `YOLOv8n` `XGBoost` `SUMO` `Streamlit` `Computer Vision`

An intelligent traffic signal control system that combines **computer vision**, **machine learning**, and **traffic simulation** to optimize green signal timing based on real-time vehicle detection from video feeds.

**Highlights**
- Real-time **YOLOv8n-based vehicle detection** with centroid tracking
- **XGBoost classifier** for dynamic signal-time prediction (`30s / 60s / 90s / 120s`)
- Multi-class recognition for **cars, buses/trucks, and bikes**
- **Weather-aware** recommendations using rain condition input
- **SUMO validation** showing ~50% reduction in waiting time vs fixed-time control
- **Interactive Streamlit dashboard** for monitoring and simulation comparison
